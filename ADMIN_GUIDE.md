## Вход в систему

Для доступа к админ-панели перейдите по адресу `/admin` (например, `site.ru/admin`).

> **Скриншот:** *[Здесь должно быть изображение — форма входа с полями «Логин» и «Пароль», кнопка «Войти»]*

Введите ваши учетные данные и нажмите «Войти». При успешной авторизации вы попадете на главную страницу панели управления.

---

## Главная (Дашборд)

После входа открывается дашборд с общей статистикой:

- **Документов** — общее количество загруженных файлов
- **Блоков** — количество редактируемых секций сайта
- **Опубликовано** — количество видимых документов на сайте
- **Скрыто** — количество скрытых документов

> **Скриншот:** *[Здесь должно быть изображение — карточки с цифрами статистики и кнопками быстрых действий]*

### Быстрые действия

На дашборде доступны кнопки:
- **Загрузить документ** — открыть форму загрузки файла
- **Редактировать блоки** — перейти к редактированию контента
- **Посмотреть сайт** — открыть главный сайт в новой вкладке
- **Синхр. с FTP** — синхронизировать документы с FTP-сервером

## Редактирование блоков

Раздел «Блоки» позволяет изменять содержимое разделов сайта.

### Выбор блока

В левой панели перечислены доступные блоки:
- Главный баннер (Hero)
- О нас
- Преимущества
- Курсы
- Контакты
- Подвал (Footer)

> **Скриншот:** *[Здесь должно быть изображение — боковая панель со списком блоков, выделен активный блок]*

Нажмите на нужный блок для редактирования.

### Редактирование контента

> **Скриншот:** *[Здесь должно быть изображение — основная форма редактирования с полями заголовка, подзаголовка, описания]*

Вкладка **«Основное»** содержит:
- **Заголовок** — название блока
- **Подзаголовок** — дополнительный текст
- **Описание / Контент** — основной текст блока
- **Текст кнопки** и **Ссылка кнопки** (для главного баннера)
- **Изображение** — загрузка или указание URL картинки

> **Скриншот:** *[Здесь должно быть изображение — поле загрузки изображения с превью]*

### Редактирование элементов

Для блоков «Преимущества», «Курсы», «О нас» и «Подвал» доступна вкладка **«Элементы»**:

> **Скриншот:** *[Здесь должно быть изображение — список элементов с кнопкой добавления нового]*

Здесь можно:
- Добавлять новые элементы (+ Добавить элемент)
- Заполнять заголовок, описание, цену (для курсов)
- Удалять элементы

### Настройки блока

> **Скриншот:** *[Здесь должно быть изображение — переключатель «Показывать блок»]*

Вкладка **«Настройки»** позволяет:
- Включить/выключить отображение блока на сайте

### Сохранение изменений

> **Скриншот:** *[Здесь должно быть изображение — кнопки «Предпросмотр» и «Сохранить», индикатор сохранения]*

Нажмите **«Сохранить»** для применения изменений. Также работает автосохранение каждые 30 секунд при наличии несохраненных изменений.

## Управление документами

Раздел «Документы» предназначен для загрузки и управления файлами.

### Загрузка документов

> **Скриншот:** *[Здесь должно быть изображение — кнопка «Загрузить документ» и поле поиска]*

Нажмите **«Загрузить документ»** или просто перетащите файлы в область загрузки (drag & drop).

Поддерживаемые форматы: PDF, DOC, DOCX, JPG, PNG

### Поиск и фильтрация

> **Скриншот:** *[Здесь должно быть изображение — поле поиска и выпадающий список фильтра]*

- **Поиск** — введите название для быстрого поиска
- **Фильтр** — показывать все / только видимые / только скрытые

### Управление документами

> **Скриншот:** *[Здесь должно быть изображение — таблица документов с колонками: название, файл, размер, статус, раздел, действия]*

Для каждого документа доступны действия:
- **Переименование** — нажмите на название для редактирования
- **Переключение видимости** — показать/скрыть на сайте
- **Скачать** — загрузить файл на устройство
- **Удалить** — удалить документ

### Массовые действия

> **Скриншот:** *[Здесь должно быть изображение — выбранные документы с кнопками массового удаления]*

Можно выбрать несколько документов и удалить их одновременно.

## Управление разделами

Разделы используются для группировки документов на сайте.

### Создание раздела

> **Скриншот:** *[Здесь должно быть изображение — поле ввода названия и кнопка «Добавить»]*

Введите название и нажмите «Добавить».

### Редактирование раздела

> **Скриншот:** *[Здесь должно быть изображение — список разделов с кнопками видимости, редактирования и удаления]*

Для каждого раздела можно:
- **Переименовать** — нажмите на название
- **Переключить видимость** — показать/скрыть все документы раздела
- **Удалить** — удалить раздел (документы останутся без раздела)

## Смена пароля

> **Скриншот:** *[Здесь должно быть изображение — форма смены пароля с полями текущего и нового пароля]*

Перейдите в раздел «Настройки» для смены пароля:

1. Введите текущий пароль
2. Введите новый пароль (минимум 4 символа)
3. Подтвердите новый пароль
4. Нажмите «Изменить пароль»

## Дополнительные функции

### Предпросмотр блока

> **Скриншот:** *[Здесь должно быть изображение — модальное окно с preview блока]*

Кнопка «Предпросмотр» показывает, как блок будет выглядеть на сайте.

## Частые вопросы

**Как добавить новый курс?**
> Перейдите в «Блоки» → выберите «Курсы» → вкладка «Элементы» → «+ Добавить элемент»

**Как скрыть раздел с сайта?**
> Перейдите в «Разделы» → нажмите иконку глаза для нужного раздела

**Как загрузить несколько документов сразу?**
> Перетащите файлы в область загрузки в разделе «Документы» или выделите несколько файлов в проводнике при нажатии на кнопку загрузки
